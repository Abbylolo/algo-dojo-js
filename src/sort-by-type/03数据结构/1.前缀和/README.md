# 前缀和

## 一、基础前缀和

### 1. 题目类型

对于数组 `nums`，若需频繁查询区间 `[left, right]` 的元素之和。使用前缀和可以将单次查询的时间复杂度从 O(n) 降低到 O(1)。

### 2. 解题思路

> 前缀和的基本思想：预处理出一个数组 `s`，其中 `s[i]` 表示原数组前 `i` 个元素的和。

1. **预处理**: 定义 `s[0] = 0`， `s[i+1] = s[i] + nums[i]`。
   - 这样 `s[i+1]` 就代表了 `nums[0]` 到 `nums[i]` 的累加和。
2. **计算区间和**: 子数组 `nums[left...right]` 的和等于 `s[right+1] - s[left]`。
   - 例如：`nums[0...2]` 的和为 `s[3] - s[0]`。

## 二、前缀和 + 哈希表

### 1. 题目类型

统计子数组和等于 $k$ 的个数，或者子数组和能被 $k$ 整除的个数。

### 2. 解题思路

> 该技巧结合了“枚举右，维护左”的思想。

1. **枚举右**: 遍历数组，计算当前的前缀和 `s[j]`。
2. **寻找左**: 我们需要找到之前的某个前缀和 `s[i]`（其中 $i < j$），使得 \`s\[j] - s\[i] = k$。
3. **转化**: 等价于在哈希表中查找是否存在值等于 `s[j] - k` 的前缀和，并统计其出现次数。
   - **注意**: 初始化哈希表 `{0: 1}`，表示前缀和为 0 的出现次数为 1。

## 三、二维前缀和

### 1. 题目类型

计算矩阵中某个子矩形的元素之和。

### 2. 解题思路

1. **预处理**: `s[i+1][j+1]` 表示从 `(0,0)` 到 `(i,j)` 的矩形和。
   - 公式: `s[i+1][j+1] = s[i][j+1] + s[i+1][j] - s[i][j] + matrix[i][j]`
2. **计算子矩形和**: 左上角 `(r1, c1)`，右下角 `(r2, c2)`。
   - 公式: `sum = s[r2+1][c2+1] - s[r2+1][c1] - s[r1][c2+1] + s[r1][c1]`
