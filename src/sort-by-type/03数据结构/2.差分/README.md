# 差分

## 一、一维差分

差分与前缀和的关系，类似**导数**与**积分**的关系。
数组 a 的 **差分的前缀和** 就是数组 a（不变）。

### 1. 题目类型

对于原数组 a，需要频繁对某个区间 \[l, r] 的所有元素进行加（或减）val 操作，最后求数组的最终状态。

### 2. 解题思路

> 一维差分的核心思想：将区间操作转化为两个端点的操作，从而将时间复杂度从 O(n) 降为 O(1)。

1. **构建差分数组 d**：
   - 定义 `d[0] = a[0]`；
   - 对于 `i > 0`，有 `d[i] = a[i] - a[i-1]`。
2. **区间加操作**：要对区间 `[l, r]` 加 val，只需：
   - `d[l] += val`（表示从 l 开始，后面的数都多了 val）；
   - `d[r+1] -= val`（表示到 r+1 开始，后面的数把刚才多的 val 减回去，抵消影响）。
3. **还原最终数组**：对差分数组求前缀和，得到最终数组。

## 二、二维差分

### 1. 题目类型

对于原矩阵 a，需要频繁对某个子矩形 `(x1, y1)` 到 `(x2, y2)` 的所有元素进行加（或减）val 操作，最后求矩阵的最终状态。

### 2. 解题思路

1. **构建差分数组 d**：
   - 二维差分是一维差分的扩展，利用“容斥原理”实现子矩阵操作。
2. **子矩阵加操作**：要对矩形左上角 `(x1, y1)`、右下角 `(x2, y2)` 加 val，只需：
   - `d[x1][y1] += val`；
   - `d[x1][y2+1] -= val`；
   - `d[x2+1][y1] -= val`；
   - `d[x2+1][y2+1] += val`。
3. **还原最终矩阵**：对差分数组求二维前缀和（先按行求前缀和，再按列求前缀和），得到最终矩阵。
